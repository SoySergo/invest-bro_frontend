# О сервисе InvestBro

> Единая европейская платформа, где бизнес, инвестиции и таланты встречаются.

---

## 1. Миссия

InvestBro — это инновационная площадка для европейского рынка, которая объединяет предпринимателей, инвесторов и специалистов в одном удобном сервисе. Мы делаем процесс покупки/продажи бизнеса, поиска инвестиций и найма таким же простым и приятным, как просмотр любимого сериала.

**Мы не брокер. Мы не краудфандинг. Мы не берём обязательств.**
Мы — площадка нового поколения для коннекта сторон.

---

## 2. Проблемы, которые мы решаем

### 2.1 Для продавцов бизнеса

| Проблема | Как мы решаем |
|----------|--------------|
| Нет единой площадки для Европы — объявления разбросаны по десяткам локальных сайтов | Одна платформа на 8 языках, покрывающая всю Западную Европу |
| Существующие доски объявлений устарели визуально и функционально | Премиальный UI с аналитикой, графиками, markdown-описаниями |
| Сложно показать реальную ценность бизнеса — только текст и цена | Финансовые метрики, чарты выручки/прибыли, загрузка документов |
| Нет фильтрации нерелевантных обращений | Верифицированные покупатели, структурированные данные, чат |
| Не понятно, кто смотрит объявление | Дашборд продавца: просмотры, избранное, контакты |

### 2.2 Для покупателей и инвесторов

| Проблема | Как мы решаем |
|----------|--------------|
| Невозможно сравнить бизнесы по метрикам | Стандартизированные карточки с revenue, profit, payback period |
| Нет инструмента поиска по конкретным параметрам | Advanced-фильтры: страна, город, категория, ценовой диапазон, выручка, тип |
| Трудно оценить реальные показатели перед контактом | Встроенные графики, финансовая аналитика на карточке |
| Нет единого каталога инвесторов по Европе | Раздел «Инвесторы» с профилями, фильтрами по стадии, чеку, отрасли |
| Рискованно связываться с непроверенными продавцами | Система верификации, рейтинги, история активности |

### 2.3 Для стартапов

| Проблема | Как мы решаем |
|----------|--------------|
| Негде показать проект на ранней стадии | Стартап-хаб: от идеи до seed-стадии |
| Поиск первых сотрудников — невозможная задача для стартапа без бюджета | Раздел вакансий с опциями: за долю, стажировка, fractional |
| Трудно найти со-основателя с нужными компетенциями | Фильтры по навыкам, опыту, формату участия |
| Инвесторы не видят стартап среди шума | Категоризация по стадии, отрасли; matching с профилями инвесторов |

### 2.4 Для ищущих работу в стартапах

| Проблема | Как мы решаем |
|----------|--------------|
| Обычные job-порталы не заточены под стартапы | Отдельный раздел с фокусом на стартап-вакансии и уникальные форматы |
| Нет прозрачности — не понятно, на какой стадии стартап | Вакансия привязана к проекту с метриками |
| Хочется стажировку в реальном проекте, а не учебном | Реальные проекты, реальные задачи, прозрачные условия |
| Fractional CXO / advisory — нигде не ищут | Специальные форматы: fractional, advisory, co-founder |

### 2.5 Для франчайзи

| Проблема | Как мы решаем |
|----------|--------------|
| Нет прозрачных данных о доходности франшизы | Метрики и отзывы на платформе |
| Непонятно, что входит в пакет и какие реальные затраты | Структурированные описания с разбивкой по статьям |
| Нет единого каталога франшиз по Европе | Отдельный раздел с фильтрами по инвестициям, категории, стране |

## 3. Пять разделов платформы

### 3.1 Маркетплейс бизнеса (Основной раздел)

**URL**: `/listings`

Покупка и продажа готового бизнеса — онлайн и офлайн.

**Содержание**: 200+ подкатегорий, организованных иерархически (см. `categories.md`):
- **Онлайн-бизнес**: e-commerce, SaaS, контент/медиа, маркетплейсы, EdTech, финтех, мобильные приложения, агентства, игровая индустрия
- **Офлайн-бизнес**: HoReCa, ритейл, услуги B2C, недвижимость, производство, оптовая торговля, логистика, агросектор, развлечения, туризм, B2B-услуги

**Карточка листинга содержит**:
- Название, описание (Markdown)
- Категория + подкатегория (иерархические badge)
- Цена продажи (EUR/GBP/CHF)
- Локация (страна, город) или Remote/Global для онлайн
- Финансовые метрики: годовая выручка, чистая прибыль, срок окупаемости
- Графики: тренд выручки по месяцам, активные пользователи (для онлайн)
- Галерея изображений
- Информация о продавце
- Кнопки: «В избранное», «Связаться»

**Фильтры**:
- Тип (онлайн / офлайн / все)
- Категория → подкатегория (каскадный)
- Страна → город
- Ценовой диапазон (slider)
- Диапазон выручки (slider)
- Срок окупаемости
- Сортировка: цена, дата, выручка, популярность

### 3.2 Инвесторы

**URL**: `/investors`

Каталог инвесторов и инвестиционных структур.

**Типы инвесторов** (по `categories.md`, раздел 2):
- Бизнес-ангелы (по стадии, отрасли, географии, типу участия)
- Венчурные фонды (по стадии, специализации, структуре)
- Частные инвесторы (HNW / UHNW, семейные офисы)
- Стратегические инвесторы (корпорации, холдинги)
- Институциональные фонды (государственные, международные, грантовые)
- Альтернативные: акселераторы, инкубаторы, долговые инструменты

**Профиль инвестора содержит**:
- Тип инвестора
- Средний чек инвестиции (диапазон)
- Стадия (pre-seed → growth)
- Отраслевая специализация (множественный выбор)
- Географический фокус
- Инструмент входа (доля, SAFE, займ, опцион)
- Требования к команде и метрикам
- Тип участия (пассивный, smart money, операционный)
- Портфельные проекты (референсы)
- Условия выхода
- Контактная информация

**Фильтры**:
- Тип инвестора
- Стадия
- Отрасль
- Средний чек (диапазон)
- Страна / регион
- Тип участия

### 3.3 Стартап-хаб

**URL**: `/listings` (фильтр по категории «Стартапы»)

Витрина для проектов на ранней стадии — от идеи до seed.

**Стадии стартапов** (по `categories.md`, раздел 1.4):
- Идея + MVP
- MVP + первые пользователи
- Ранняя выручка (pre-seed)
- Seed-стадия (с трекшном)

**Карточка стартапа** (расширенная карточка листинга):
- Стадия проекта
- Размер ищемых инвестиций
- Equity, предлагаемое инвестору
- Текущие метрики (MAU, MRR, рост)
- Состав команды
- Tech-стек
- Pitch-описание (Markdown)

### 3.4 Найм и таланты

**URL**: `/jobs`

Вакансии в стартапах, малом и среднем бизнесе.

**Категории ролей** (по `categories.md`, раздел 3):
- **Для стартапов**: продукт и технологии, рост и маркетинг, продажи, операции и финансы, HR, специфика стартапа (со-основатель, стажёр, adviser, fractional CXO)
- **Для зрелого бизнеса**: топ-менеджмент, среднее звено, специалисты, производственный персонал, сервисный персонал
- **Для онлайн-бизнесов и агентств**: проджект-менеджеры, трафик, разработка, контент, аналитика

**Карточка вакансии содержит**:
- Название позиции
- Компания / проект (ссылка на листинг, если есть)
- Описание (Markdown)
- Категория роли
- Уровень (Junior / Middle / Senior / Lead / Head / C-level)
- Формат занятости: full-time, part-time, project, freelance, стажировка, fractional, remote, гибрид, офис
- Локация (или Remote)
- Зарплатная вилка (опционально)
- Опционная программа (для стартапов)
- Требуемый опыт (лет)
- Требуемый стек / инструменты
- Языки работы
- Срочность найма

**Фильтры**:
- Категория роли
- Уровень
- Формат занятости
- Локация / Remote
- Зарплатный диапазон
- С опционами / без
- Стажировка (оплачиваемая / неоплачиваемая)

### 3.5 Франшизы и партнёрство

**URL**: `/franchises`

Продажа франшиз, поиск партнёров, продажа долей.

**Типы** (по `categories.md`, разделы 1.3 и 1.5):
- Продажа собственной франшизы
- Продажа действующей точки франшизы
- Мастер-франшиза (региональный франчайзи)
- Продажа доли в бизнесе (миноритарная / мажоритарная)
- Поиск партнёра с вхождением в долю
- Продажа активов (без юр. лица)

**Карточка франшизы**:
- Бренд / название
- Описание (Markdown)
- Категория бизнеса
- Стоимость входа (паушальный взнос)
- Роялти (% или фиксированная сумма)
- Требуемые инвестиции (общие)
- Количество действующих точек
- Срок окупаемости
- Поддержка франчайзера (обучение, маркетинг, поставки)
- Локация (где открыть)
- Территориальная эксклюзивность

### 6.3 Детализация ключевых страниц

#### Главная (`/`)

Структура сверху вниз:

1. **Hero-секция**: полноэкранный gradient-блок, заголовок, подзаголовок, 2 CTA кнопки («Смотреть бизнесы» + «Разместить объявление»), поисковая строка с быстрым поиском
2. **Как это работает**: 3 шага для покупателей + 3 шага для продавцов, анимированные иконки
3. **Trending бизнесы**: горизонтальный scroll-ряд (как Netflix) — 6 карточек самых просматриваемых
4. **По категориям — Онлайн**: горизонтальный scroll-ряд с иконками категорий
5. **По категориям — Офлайн**: горизонтальный scroll-ряд с иконками категорий
6. **Ищете инвестиции?**: CTA-блок со ссылкой на каталог инвесторов
7. **Вакансии в стартапах**: горизонтальный scroll-ряд последних вакансий
8. **Франшизы**: горизонтальный scroll-ряд
9. **Статистика платформы**: анимированные счётчики (листинги, инвесторы, страны, вакансии)
10. **CTA финал**: «Готовы начать?» + кнопки регистрации

#### Каталог бизнесов (`/listings`)

1. **Верхний блок**: заголовок + подзаголовок + общее количество
2. **Панель фильтров** (слева на desktop, sheet на мобильных):
   - Поиск (текст)
   - Тип (онлайн / офлайн / все) — радио
   - Категория → подкатегория — каскадный select
   - Страна → город — каскадный select
   - Ценовой диапазон — dual slider
   - Годовая выручка — dual slider
   - Сортировка — select (цена ↑↓, дата, выручка, популярность)
   - Кнопка «Сбросить фильтры»
3. **Сетка карточек**: 3 колонки (desktop), 2 (tablet), 1 (mobile)
4. **Пагинация**: page-based или infinite scroll
5. **Пустое состояние**: иллюстрация + текст + ссылка на создание

#### Детальная страница листинга (`/listing/[id]`)

**Левая колонка (2/3)**:
1. Хлебные крошки (Home → Category → Subcategory → Title)
2. Badges: тип (онлайн/офлайн), категория, страна, статус
3. Заголовок (h1)
4. Карусель изображений (если есть)
5. Обзор бизнеса (Markdown → HTML)
6. Финансовые показатели:
   - Revenue / Profit / Payback — большие цифры
   - График тренда выручки по месяцам (AreaChart)
   - График пользователей (для онлайн)
7. Локация на карте (будущее — интеграция с картами)

**Правая колонка (1/3 — sticky sidebar)**:
1. Asking Price — крупная цифра
2. Ключевые метрики (компактно)
3. Информация о продавце (аватар, имя, дата регистрации, количество листингов)
4. Кнопка «Связаться с продавцом» (открывает чат)
5. Кнопка «В избранное»
6. Блок безопасности: предупреждение о проверке перед сделкой
7. Share-кнопки (копировать ссылку, соцсети)

#### Wizard создания листинга (`/listing/create`)

6 шагов с progress bar:

1. **Тип и категория**: выбор онлайн/офлайн → категория → подкатегория (каскадный)
2. **Основная информация**: название, описание (Markdown-редактор), цена, валюта (EUR/GBP/CHF)
3. **Локация**: страна (select) → город (select или текст); для онлайн — чекбокс «Remote / Global»
4. **Финансовые метрики**: годовая выручка, чистая прибыль, данные по месяцам (таблица 12 месяцев — опционально)
5. **Изображения**: drag-and-drop загрузка до 10 фото, сортировка порядка, обложка
6. **Проверка и публикация**: предпросмотр карточки, кнопка «Опубликовать» / «Сохранить как черновик»

## 7. Уникальные конкурентные преимущества

1. **Единая экосистема**: бизнес + инвестиции + найм + франшизы — в одном месте, а не на 4 разных сайтах
2. **Европейский фокус**: 8 языков, EUR как базовая валюта, европейские страны
3. **Финансовая аналитика на карточке**: графики выручки и прибыли видны до клика — экономит время
4. **Дизайн уровня Netflix**: тёмная тема, glass-morphism, плавные анимации — не похоже на типичную доску объявлений
